## 🎯 这是什么

一个**纯前端**实现的德州扑克（Texas Hold'em）概率计算工具。

输入你的手牌 + 已知的公共牌，自动计算你能击中的**前7个顶级牌型概率**，帮助你快速判断当前手牌潜力和规划行动线。

## 🎮 使用说明

### 选牌方式

```
┌─────────────────────────────────────────┐
│  ♠黑桃  ♥红桃  ♣梅花  ♦方块              │  ← ① 选一个花色
├─────────────────────────────────────────┤
│  A  K  Q  J  10  9  8  7  6  5  4  3  2 │  ← ② 选一个数字
└─────────────────────────────────────────┘
```

1. **选花色**：点击4个花色按钮之一（黑桃/红桃/梅花/方块）
2. **选数字**：点击13个数字按钮之一
3. **自动组合**：花色 + 数字 = 一张牌，自动加入已选区域
4. **撤销**：点「↩️ 撤销」删除最后一张，或直接点击已选牌取消

### 四阶段流程

| 阶段 | 手牌 | 公共牌 | 阶段指示器 |
|:---|:---:|:---:|:---|
| **Pre-flop** | 2张 | 0张 | 翻牌前 |
| **Flop** | 2张 | 3张 | 翻牌圈 |
| **Turn** | 2张 | 4张 | 转牌圈 |
| **River** | 2张 | 5张 | 河牌圈 |

---

## 🧮 计算逻辑

| 阶段 | 剩余未知牌 | 计算方法 | 精确度 |
|:---|:---:|:---|:---:|
| Pre-flop | 50张 | 蒙特卡洛模拟（20万次抽样） | 近似 |
| Flop | 47张 | 精确枚举（1,081种组合） | 精确 |
| Turn | 46张 | 精确枚举（46种组合） | 精确 |
| River | 45张 | 直接计算最终牌型 | 确定 |

## 📊 牌型概率参考

在**完全随机**的7张牌中，各牌型的理论出现概率：

| 牌型 | 概率 | 约多少把出现一次 |
|:---|:---:|:---|
| 一对 | 43.82% | 每2把 |
| 两对 | 23.50% | 每4把 |
| 三条 | 4.83% | 每21把 |
| 顺子 | 4.62% | 每22把 |
| 同花 | 3.03% | 每33把 |
| 葫芦 | 2.60% | 每38把 |
| 四条 | 0.168% | 每595把 |

> 实际概率因**具体起手牌**而异。口袋对子中三条的概率远高于随机平均！

## 🛠️ 项目结构

```
texas-holdem-calculator/
├── index.html           # 主页面
├── css/
│   └── style.css        # 绿色赌桌风格样式
├── js/
│   └── poker-engine.js  # 核心引擎（牌型评估 + 概率计算）
├── test/
│   └── test-engine.cjs  # 单元测试
├── sw.js                # Service Worker（PWA离线支持）
├── manifest.json        # PWA配置
└── README.md            # 本文件
```

### 技术栈

- **纯原生 JavaScript**（ES6+）
- **纯 CSS3**（CSS Variables + Grid + Flexbox）
- **零框架、零依赖**

### 核心算法

1. **牌型评估**：从7张牌枚举 C(7,5)=21 种组合，取最强牌型
2. **概率计算**：精确枚举 / 蒙特卡洛模拟 双模式自适应
3. **结果排序**：前7个顶级牌型按概率降序排列
